# WETUBE

- NodeJS, Express, JavsScript ES6를 공부하며 만든 동영상 공유 사이트입니다.
- Github, Facebook 혹은 이메일을 통해 회원가입을 할 수 있습니다.
- 동영상을 녹화하거나 업로드할 수 있으며, 조회할 수 있습니다.
- 제목을 검색하여 동영상을 검색할 수 있습니다.
- 파일은 AWS S3로 관리하고 있으며, heroku에 배포하였습니다.
